# PopClip extension

## Champollion Deckを使った翻訳

選択した文章を2026年2月に公開されたChampollion Deckを使って翻訳します。

### 準備

下記のアプリケーションが必要なのでそれぞれのサイトを参照してインストールします。

- [ollama](https://ollama.com)
- [Champollion Deck](https://note.com/t_trace/n/n44e4dca5dd45?sub_rt=share_pb)

### インストール

ファイルChampollionDeck.popclipextzをダウンロードし、ダブルクリックするとpopclipに組み込まれます。

## 物書堂辞書用拡張

## manページをターミナルで開く

選択したコマンド名の文字列を**manページをターミナルで開く**で開く、PopClip拡張。

- manpage.popclipextzをダウンロードし、ダブルクリックしてインストールする。
